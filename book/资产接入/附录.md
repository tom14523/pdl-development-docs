## 基础错误码

| 错误码 | 说明 |
| ----- | ---- |
| 30000 | 数据不完整 |
| 40100 | 验签不通过 |
| 50001 | 商户不存在 |
| 50002 | 海投汇用户不存在 |
| 50003 | 借款用户未开户 |
| 50004 | 借款用户未设置交易密码 |
| 50005 | 借款用户未缴费授权 |
| 50006 | 借款用户不存在 |
